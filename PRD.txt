Component & UI SpecificationsThree-Panel Layout:Left: Contextual Sidebar. If navigating /sephora, the sidebar displays only Sephora links and global guides. It hides links for Ikea and other brands.Middle: Formatted markdown descriptions, parameter tables, and schema constraints.Right: Code snippets and interactive playground.Interactive API Client: Embed Fumadocs’ integrated Scalar runtime playground module on endpoint sheets, allowing developers to execute real test requests directly from the browser window.Contextual Search Modal: The global search field must isolate queries based on current user paths.
